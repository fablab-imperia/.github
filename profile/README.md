## Who we are

Fablab Imperia is a non-profit association (APS) and part of the global network of fab labs — local workshops that make invention accessible by providing tools for digital fabrication. In our lab you'll find 3D printers, a CNC mill, a vinyl cutter, an electronics bench, and mechanical tools: everything you need to turn an idea into a prototype.

We run on **peer learning** — members build things alone and in groups, teaching each other along the way. *If I make, I learn.*

## What you'll find here

This organization hosts the source code, hardware designs, and documentation for our projects. Everything is open, so anyone can use it, learn from it, and build on it.

**Featured project — [MeshBee](https://github.com/fablab-imperia/meshbee)**
Solar-powered sensor nodes for beehives that report over MQTT to a server and a mobile app — an IoT toolkit for beekeeping. It spans several repositories:
[firmware](https://github.com/fablab-imperia/meshbee-firmware) · [server](https://github.com/fablab-imperia/meshbee-server) · [app](https://github.com/fablab-imperia/meshbee-app) · [hardware](https://github.com/fablab-imperia/meshbee-hardware)

Browse all our repositories below to see what else our members are working on.

## Get involved

You don't need to be an expert — contributions of every size are welcome, from fixing a typo to designing a board.

- Read our [Contributing guide](https://github.com/fablab-imperia/.github/blob/main/CONTRIBUTING.md) to get started.
- Look for issues labelled **good first issue**.
- We expect everyone to follow our [Code of Conduct](https://github.com/fablab-imperia/.github/blob/main/CODE_OF_CONDUCT.md).
- Found a security problem? See our [Security Policy](https://github.com/fablab-imperia/.github/blob/main/SECURITY.md).

Prefer to build in person? Come to an [event](https://fablabimperia.org/eventi/) or visit the lab.
